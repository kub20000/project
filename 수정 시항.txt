
<유저 페이지 수정 사항>

- 강의 바로가기 버튼 안됨
- 로그아웃 상태일 때 인기강의 좋아요 수 x
- 로그 아웃 상태일 때 인기강의 클릭 시 강의로 넘어감 -> 로그인 페이지로 유도 하기
- 페이지마다 푸터 넣기
- 푸터 링크 수정
- 강의 영상 화면에 꽉 채우기
- 게시판 다음글 이전글 순서 바꾸기
- 대시보드 나의 레시피 최신 레시피 3개 불러오기
- 학습 기록에 좋아요한 강의 추가
- 나의 레시피 레시피 내용 정리
- 레시피 디테일 불러오기


<관리자 페이지 수정 사항>
- 화원 정보 관리 페이지
- 게시물 수정 시 작성자가 글번호로 보임
