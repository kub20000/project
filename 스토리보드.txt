스토리보드 

■ 나의 냉장고 
◇ 사용자
 : 학습자(student)
◇ 목표
 : 학습자의 냉장고에 구비되어있는 재료를 이용하여 만들 수 있는 비건 레시피를 확인한다.
◇ 주요 시나리오 
1. 학습자가 시스템에 로그인한다.
2. 헤더에 표시된 '나의 냉장고' 메뉴로 이동한다.
3. 본인의 냉장고에 구비되어 있는 재료들을 검색/태그하여 입력한다.
4. 재료들을 입력하였으면 중앙에 있는 '레시피 찾기' 버튼을 클릭한다.
◇ 화면 구성요소
1. 헤더: 로고, 메뉴(홈, 강의, 나의냉장고, 로그인, 게시판, 마이페이지)
2. 페이지 제목: 나의 냉장고
3. 재료 입력 영역:
    ㄴ 식재료 태그
    ㄴ 재료 검색창
4. 레시피 찾기
◇ 비고
 - 등록된 강의를 통해 배운 레시피 뿐 아니라 학습자가 쉽게 접할 수 있거나 그 냉장고에 구비된 재료를 사용하여 만들 수 있는 비건 요리 레시피를 알게 함으로써 학습자로 하여금 조금 더 능동적으로 비건식을 접할 수 있게 할 수 있다.

■ 나의 학습기록
◇ 사용자
 : 학습자(student)
◇ 목표
 : 자신이 수강한 강의를 다시 보고 싶을 때, 혹은 불가피하게 강의를 보다가 중간에 끊어졌을 경우 조금 더 원활한 경로로 해당 강의를 찾아 보던 부분으로 이동하게 한다. 
◇ 주요 시나리오 
1. 학습자가 시스템에 로그인한다.
2. 헤더에 표시된 마이페이지로 접속한다.
3. 마이페이지 접속 시 노출되는 메뉴 중 최근 강의 탭을 클릭한다.
4. 최근 내가 본 강의의 진도율이 바(bar)형식으로 표시된다.
5. 이어보고싶을 경우, 이어보기 버튼을 클릭하여 끊어진 부분부터 이어서 볼 수 있다.
◇ 화면 구성요소
1. 헤더: 로고, 메뉴(홈, 강의, 나의냉장고, 게시판, 마이페이지)
2. 페이지 제목: 나의 학습기록
3. 서브메뉴(내 정보 수정, 내 학습기록, 나의 레시피, 내 게시글 관리, 회원탈퇴)
4. 카테고리별 학습진도율
5. 최근 본 강의:
  - 강의카드
     ㄴ 강의명
     ㄴ 진도율 표시
     ㄴ 마지막 시청 날짜
6. 완료한 강의:
  - 강의카드
    ㄴ 진도율
◇ 비고
  : 진도율을 시각적으로 보여주어 학습자에게 진도 현황을 확인시켜줄 수 있다.

■ 게시판 
◇ 사용자
 : 학습자(student), 관리자(admin)
◇ 목표
 : 단순히 강의 시청으로 그치는 것이 아니라 비건식에 관심을 갖거나 도전하고 있는 학습자들끼리 강의를 보고 배운 점이나 직접 비건식 레시피를 시연한 것, 흑은 유용한 정보 등을 공유할 수 있게하며, 해당 취지에 부적절한 내용이 있을 경우 관리자가 수정/삭제 등을 하여 관리하게 한다.
◇ 주요 시나리오 
1. 학습자/관리자가 시스템에 로그인한다.
2. 헤더에 표시된 게시판으로 접속한다.
3. 학습자는 비건식에 대한 글 쓰기, 혹은 이미 게시된 글을 확인하며 비건식에 대한 정보를 공유한다.
3-1. 관리자는 게시글을 확인하여 부적절한 게시글이 있는지 확인하여 수정/삭제 하거나 글쓰기를 사용하여 공지사항을 공지, 혹은 쓰고자 하는 글을 게시한다.
4. 필요에 따라선 본인이 게시했던 글을 수정/삭제한다.

◇ 화면 구성요소
1. 헤더: 로고, 메뉴(홈, 강의, 나의냉장고, 게시판, 마이페이지, 로그아웃)
2. 페이지 제목: 자유게시판
3. 검색창
4. 다음 페이지
◇ 비고

■ 나의 레시피
◇ 사용자
 : 학습자(student)
◇ 목표
 : 나의 냉장고를 통해 확인한 레시피를 저장하여 나중에도 간편하게 확인할 수 있게 한다.
◇ 주요 시나리오 
1. 학습자가 시스템에 로그인한다.
2. 헤더에 표시된 마이페이지로 접속한다.
3. 좌측 서브 메뉴에 확인되는 나의 레시피를 클릭한다.
4. 내가 저장한 레시피들을 확인한다.

◇ 화면 구성요소
1. 헤더: 로고, 메뉴(홈, 강의, 나의냉장고, 게시판, 마이페이지, 로그아웃)
2. 페이지 제목: 나의 레시피
3. 저장한 레시피 내용
◇ 비고


■ 메인페이지 - 인기 강의
◇ 사용자
 : 학습자(student)
◇ 목표
 : 학습하는 유저들이 선호하는 강의들을 메인 페이지에서 노출시킴으로써, 처음 해당 사이트를 접한 학습자들이 강의에 대한 접근을 완화한다. 
◇ 주요 시나리오 
1. 학습자가 시스템에 로그인한다.
2. 마우스 스크롤을 내려 인기강의를 확인한다.
3. 인기 강의 중 관심이 있는 강의를 클릭한다.
4. 강의페이지로 넘어가서 해당 강의를 통하여 비건 라이프를 학습한다.

◇ 화면 구성요소
1. 헤더: 로고, 메뉴(홈, 강의, 나의냉장고, 게시판, 마이페이지, 로그아웃, 마이페이지)
2. 페이지 제목: 메인페이지-강의
3. 검색창
4. 검색 분야
5.  - 강의카드
     ㄴ 강의명
     ㄴ 진도율 표시
     ㄴ 강의 분야
     ㄴ 강의재생 버튼
     ㄴ 퀴즈풀기
     ㄴ 좋아요 버튼
◇ 비고
- 강의 100% 수강 후 퀴즈를 풀 수 있다.
